# Retail_Customer_Segmentation_and_Analysis_Using-RFM

**Project Summary**

This project focuses on customer segmentation using RFM analysis and machine learning to understand customer behavior and support data-driven business decisions.


#### Interactive App: https://retail-customer-segmentation-and-analysis-using-rfm.streamlit.app/

#### Interactive App: https://www.kaggle.com/code/mohammadelfar/retail-customer-segmentation-analysis-using-rfm


## Workflow overview:

#### 1️⃣ Data Understanding & Preparation
**Cleaned the data, handled missing values, and ensured data quality.**

#### 2️⃣ Univariate Analysis
**Performed EDA on individual features to understand distributions and detect issues.**

#### 3️⃣ RFM Table Creation & EDA
**Built RFM metrics (Recency, Frequency, Monetary) and analyzed their distributions.**

#### 4️⃣ RFM & Business Analysis
**Combined RFM features with original data to extract behavioral insights.**

#### 5️⃣ Modeling & Clustering
**Applied K-Means clustering with 3 customer segments based on RFM features.**

#### 6️⃣ Cluster Profiling & Interpretation
**Interpreted each cluster and identified high-value, loyal, and low-engagement customers.**

#### 7️⃣ Deployment
**Deployed an interactive dashboard using Streamlit to visualize segments and insights.**
